# 🚀 Antigravity Portfolio

A premium, immersive developer portfolio built with **Next.js**, **TypeScript**, and **Tailwind CSS**. Inspired by high-end interactive experiences, this portfolio features smooth scrolling, cinematic animations, and a modern UI.

## 🌐 Live Demo

## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/SHITALILAPATE/antigravity-portfolio.git
```

Go into the project

```bash
cd antigravity-portfolio
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Open

```
http://localhost:3000
```

---

## 📦 Build

```bash
npm run build
```
